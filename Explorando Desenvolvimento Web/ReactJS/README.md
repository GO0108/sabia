
# Como rodar

## backend

na pasta backend, criar um ambiente via:


```bash 
python3 -m venv venv  
. venv/Scripts/activate
pip install -r requirements.txt
```

### rodando o backend
```python
. venv/Scripts/activate
flask run
```


## Frontend

na pasta backend, criar um ambiente via:


```bash 
npm i
```

### gerando a extensão
```node
npm run build
```
### rodando o Frontend via localhost
```node
npm run start
```
